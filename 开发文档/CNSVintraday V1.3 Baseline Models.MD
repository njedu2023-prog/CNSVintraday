# CNSVintraday V1.3 Baseline Models

版本：

V1.3

阶段：

Baseline Models

前置条件：

V1.1.1 Data Connector 已通过

V1.2 Feature Engine 已通过

--------------------------------------------------

# 一、开发目标

V1.3 的目标：

建立盘中线第一套可验证预测系统。

注意：

V1.3 不是最终模型。

V1.3 是基准模型层。

用于回答：

Feature 是否有预测能力？

模型是否真正优于随机？

未来高级模型是否真的提升？

--------------------------------------------------

# 二、核心原则

原则1

所有模型必须可解释。

--------------------------------------------------

原则2

所有模型必须Replay验证。

--------------------------------------------------

原则3

必须保留最简单模型。

禁止只保留高级模型。

--------------------------------------------------

原则4

所有预测均为观察级预测。

禁止：

买入信号

卖出信号

自动交易

--------------------------------------------------

# 三、预测目标

预测对象：

T日14:00

预测

T+1交易日

--------------------------------------------------

输出：

上涨概率

下跌概率

预期涨跌幅

--------------------------------------------------

不是：

交易信号

--------------------------------------------------

# 四、标签定义

建立统一标签。

label_up_t1

定义：

T+1收盘价 > T日14:00价格

=1

否则

=0

--------------------------------------------------

label_return_t1

定义：

(T+1收盘价 / T日14:00价格)-1

--------------------------------------------------

禁止：

未来信息进入Feature。

--------------------------------------------------
