# CNSVintraday Decision JSON Outputs

`data/decision/` is the V2.0 intraday decision report JSON output directory.

Generated files:

- `data/decision/intraday_decision_latest.json`
- `data/decision/history/YYYYMMDD/intraday_decision_1400.json`

Generated artifacts are not always committed, but the pipeline must be able to create them.
