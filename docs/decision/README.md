# CNSVintraday Decision HTML Outputs

`docs/decision/` is the V2.0 intraday decision report HTML output directory.

Generated files:

- `docs/decision/intraday_decision_latest.html`
- `docs/decision/reports/YYYYMMDD/intraday_decision_1400.html`

Generated artifacts are not always committed, but the pipeline must be able to create them.
